Übung: Dynamische Web-Elemente

Erledige die folgenden Aufgaben, um deine Fähigkeiten in JavaScript zu trainieren. Du musst alle Buttons, Eingabefelder und JavaScript-Funktionen selbst erstellen.

1. Der Rezept-Anzeiger

1.1. Erstelle zwei Buttons mit den Aufschriften "Pfannkuchen" und "Spaghetti Carbonara".
1.2. Schreibe eine JavaScript-Funktion `zeigePfannkuchen()`, die im Div mit der ID rezept-anzeige ein einfaches Pfannkuchen-Rezept anzeigt.
1.3. Schreibe eine zweite Funktion `zeigeSpaghetti()`, die das Rezept für Spaghetti anzeigt.
1.4. Verknüpfe die Buttons so, dass beim Klick das jeweilige Rezept angezeigt wird. Der Inhalt des Divs soll dabei immer komplett ersetzt werden.

2. Die interaktive Einkaufsliste

2.1. Erstelle drei Buttons für die Zutaten "Mehl", "Eier" und "Milch".
2.2. Schreibe eine einzige JavaScript-Funktion namens `fuegeZutatHinzu(zutat)`. Diese Funktion soll eine Zutat als Parameter entgegennehmen.
2.3. Die Funktion soll die übergebene Zutat zum Div mit der ID einkaufsliste hinzufügen. Jede neue Zutat soll in einer neuen Zeile erscheinen.
2.4. Erstelle einen weiteren Button "Liste leeren". Dieser soll eine Funktion `leereListe()` aufrufen, die den gesamten Inhalt der Einkaufsliste löscht.

3. Der dynamische Benutzer-Avatar

Am Anfang des Scripts ist bereits eine Variable let benutzername = 'Gast'; deklariert.
3.1. Erstelle einen Button "Avatar anzeigen". Bei Klick soll eine Funktion `zeigeAvatar()` aufgerufen werden.
3.2. Die Funktion zeigeAvatar() soll im Div mit der ID avatar-container ein neues Div mit der Klasse avatar erstellen. In diesem Div soll der aktuelle Wert der Variable benutzername angezeigt werden.
3.3. Erstelle zwei weitere Buttons, "Alex" und "Maria". Bei Klick auf einen dieser Buttons soll der Wert der Variable benutzername entsprechend geändert und der Avatar sofort aktualisiert werden.

4. Das Gästebuch

4.1. Erstelle ein Text-Eingabefeld für einen Namen (gib ihm die ID input-name) und ein weiteres für eine Nachricht (ID input-nachricht).
4.2. Erstelle einen "Senden"-Button.
4.3. Schreibe eine Funktion eintragHinzufuegen(), die durch den Button aufgerufen wird.
4.4. Diese Funktion soll den Text aus beiden Eingabefeldern auslesen.
4.5. Füge dem Div mit der ID gaestebuch einen neuen Eintrag hinzu. Der Eintrag soll so formatiert sein: <b>[Name]</b> schrieb: <i>[Nachricht]</i><br>.
4.6. Sorge dafür, dass die Eingabefelder nach dem Senden automatisch geleert werden.
4.7. (Bonus) Verhindere durch eine if-Abfrage, dass eine leere Nachricht gesendet werden kann.
